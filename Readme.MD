## Create Microservice in NodeJS

# Service-1

    [+] : User Service : [Include Create users and Record Post By Users]
    [+] : Run On port : 5000

# Service-2

    [+] : Post Service : [Include create post && Saves All Posts]
    [+] : Run On port : 5001

# Api-Gateway

    [+] : records all services
    [+] : Run On port : 8001

# How to run

```bash
git clone
cd microservice/microservices
cd apigateway && npm i
cd User && npm i
cd Post && npm i
```
